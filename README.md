# Bogle Payment Portal

A React + Tailwind CSS demo that mimics Stripe’s polished checkout UI.

## ✨ Features

- Vite + React 18 fast-refresh development
- Tailwind CSS utility-first styling
- Responsive, accessible form components
- Fake async payment handler for demo purposes

## 🚀 Getting Started

```bash
# install dependencies
npm install

# start dev server
npm run dev

# build production assets
npm run build
```

Open http://localhost:5173 in your browser.
